# Assembly Plan — Milestone 1

## Stage A: print validation coupons

1. Print the connector-panel coupon in the intended PETG-CF profile.
2. Test Powerpole, USB-C, switch, and display fit.
3. Record dimensional corrections in an issue.
4. Print an M3 heat-set-insert coupon with at least three pilot diameters.
5. Select the pilot diameter that permits straight installation without wall cracking.

## Stage B: mechanical dry assembly

1. Install heat-set inserts in the body, lid, sled, and panel retainers.
2. Fit blank front and rear panels.
3. Assemble the lid without electronics.
4. Check seam consistency and verify that screws do not bottom out.
5. Fit non-slip feet and test both strap channels.
6. Verify AMPS adapter alignment.

## Stage C: component-fit assembly

1. Install panel connectors without wiring.
2. Confirm rear clearance and tool access.
3. Install the fuse block and PD module mockups on the sled.
4. Route dummy wire or cord through all intended channels.
5. Verify lid closure with service loops present.

## Stage D: acceptance checks

- No visible cracking around inserts
- No panel movement under firm hand pressure
- No connector interference with the lid or sled
- USB-C cable retention resists a 20 N axial pull without disconnecting the panel interface
- Enclosure rests flat on all four feet
- 25 mm strap can be installed without opening the enclosure
- Branding is legible and visually centered

Electrical installation and energized testing are outside Milestone 1.
